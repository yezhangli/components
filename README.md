##开始使用
1.添加css样式
    使用本框架前，请在css中开启border-box
    ...
    *,*::before,*::after{box-sizing: border-box;}
    ...
    还需要设置默认颜色等变量，后续会更改为scss变量
    ...
    :root{
        --button-height:32px;
        --font-size:14px;
        --button-bg:white;
        --button-active-bg:#eee;
        --border-radius:4px;
        --color:#333;
        --border-color: #999;
        --border-color-hover:#666;
    }
    IE15及以上浏览器都支持该样式
